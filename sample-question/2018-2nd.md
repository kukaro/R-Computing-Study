13번 문제
rep(1,3) : 1을 3번 반복 
-> 1 1 1
seq(1,5,by=2) : 1에서 5까지 2간격으로
-> 1 3 5
rev(seq(1,5,length=3)) : 1에서 5까지 3칸짜리 벡터로 간격 유지, 그다음 반대로
-> 5 3 1
rep(2,3) : 2를 3번 반복
-> 2 2 2
결과  : 1 1 1 1 3 5 5 3 1 2 2 2
1이 5개이다 -> (1+2+3+4+5)/5 = 3
2가 3개이다 -> (6+7+8)/3 = 7
3이 2개이다 -> (9+10)/2 = 9.5
5가 2개이다 -> (11+12)/2 = 11.5
가 -> 9.5
나 -> 9.5
참고 - order함수도 공부해야함
